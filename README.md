Carte 3D à retournement au survol

Ce projet propose la création d'une Carte 3D de Profil qui intègre un mécanisme de retournement.

- Aperçu du Projet

Il s'agit d'une Carte 3D à retournement au survol. La particularité de cette réalisation est qu'elle est conçue uniquement en HTML et CSS.

- Technologies et Concepts Clés
  
Le projet est basé sur l'utilisation stricte de:
• HTML
• CSS
De plus, cette conception s'aligne sur les principes de la Conception web adaptative, qui est également désignée par le terme anglais Responsive Web Design.
